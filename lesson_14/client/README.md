## Getting Started

### Prerequisites

- Node.js (18+ recommended)
- Bun (or npm/yarn)

### Installation

```bash
# Install dependencies
bun install
# or
npm install
```

### Development

```bash
# Start the dev server
bun run dev
# or
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build

```bash
bun run build
# or
npm run build
```
